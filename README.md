# WELL At-Scale Performance Dashboard

An interactive building wellness assessment and benchmarking tool built on the WELL At-Scale Standard framework by IWBI.

## About
This dashboard was built to simulate the core analytical workflow of a WELL At-Scale assessment — scoring buildings across 10 concept areas, visualizing performance gaps, benchmarking against industry standards, and surfacing recommendations for leadership.

## Features
- Score 1-3 buildings across all 10 WELL concept areas
- Radar chart, heatmap, and gauge visualizations
- Gap analysis vs industry benchmark with recommendations
- Priority matrix showing impact vs implementation effort
- Custom target score setting
- Notes field per concept area
- Historical trend tracking
- Export to .txt report and .csv
- Shareable URL — encode full assessment in a link
- Print / PDF export

## How to Use
